== Description ==

Quickly search and filter your Products and edit Stock Quantities very fast!
This is a very new plugin, please let us know your requests and suggestions in the Support page. We plan to add many functions in the coming week(s)!

== Installation ==

* Login as Administrator of your portal;
* Navigate to Plugins -> Add New;
* Click on Upload;
* Select and Upload the .zip file you downloaded from this website;
* Activate the new plugin.
